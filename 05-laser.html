Laser Cutting

This assignment was to create a laser printed musical box. First is to make a 3D render of the box in Fusion 360 to visualise the box with exact dimensions. Ensure that each side is its own body and export each body as DXF.

Import each DXF into Inkscape to create a 2D layout for the laser cutting machine to understand and cut.
